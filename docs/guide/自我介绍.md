# 项目表述

### 仓储管理系统

那我最近的一个项目，是一个企业仓储管理系统，里面包含标签、库存、采购、生产、销售管理等一系列功能，app则包含扫码入库、质检、上架、出库、过账等，工作人员通过扫描标签二维码的方式对药品进行信息化管理，极大地提高了仓储管理效率。那我在项目里面主要是负责跟前端app和web对接，跟企业的SAP系统对接，标签打印、日志、部署上线等等工作。标签打印是用了一个报表软件画的模板，然后在后台填充数据生成PDF文件传递给前端进行打印，日志是使用Spring AOP实现的。
